# Sobre o Projeto: Dashboard de E-commerce #

Desenvolvi um dashboard no Power BI focado no comportamento de vendas, integrando bases transacionais e cadastrais via SQL.Para garantir que a análise fosse 100% confiável, utilizei o INNER JOIN na unificação das tabelas. 

O foco principal foi gerar insights práticos. O dashboard destaca métricas essenciais como receita total, ticket médio e volume de clientes, além de permitir quebras detalhadas por categoria, gênero, região e método de pagamento.
